## ğŸ”¹ How It Works

1. **Dataset preprocessing:** Email text is cleaned and processed.  
2. **Feature extraction:** Text is converted into numerical features using TF-IDF.  
3. **Model training:** Logistic Regression classifier is trained on the dataset.  
4. **Prediction:** User can input any email and the model predicts phishing or safe.  
5. **Web interface:** Streamlit provides a simple interface to test emails.
